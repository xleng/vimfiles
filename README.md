vimfiles
========
This is based on https://github.com/gmarik/vimfiles


Useage:

Linux:

git clone https://github.com/xleng/vimfiles.git ~/.vimfiles

cd ~/.vimfiles

chmod a+x config.sh

./config.sh


Windows:

install git for windows: https://git-for-windows.github.io/

rung the following command in git bash

git clone https://github.com/xleng/vimfiles.git ~/vimfiles

cd ~/vimfiles

chmod a+x config.sh

./config.sh
